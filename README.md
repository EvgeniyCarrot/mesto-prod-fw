# Проектная работа Mesto. Работа с сервером

Проект посвящен нтерактивной клиентской части веб-приложения для обмена, добавления, удаления и ваимодействия с постами и фотографиями.
В рамках проекта, пользователи могут просматривать и лайкать посты других участников. Будучи создателем поста, участник может
добавлять свои посты и удалять их.

# Выбор варианта:

Для выполнения было выбрано личное задание из варианта 1: реализация модального окна с информацией о карточке.

## Технологический стек:

HTML5, CSS3, JavaScript
Сборщик проектов Vite
Api: https://mesto.nomoreparties.co

### Команды:

- Установка зависимостей - npm install
- Запуск локального сервера разработки: npm run dev
- Сборка проекта: npm run deploy
